vrok
====

plays FLAC, MPEG and AAC(planned)<br>
Shibatch SuperEQ<br>
Windows support and JACK support is planned<br>
Gapless, not implemented yet<br>

Compiling
---------

run these commands in the project directory

<pre>
qmake
make
</pre>

Running
-------
<pre>
./vrok
</pre>
